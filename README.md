# REST-API-Practice

## Architecture
Frontend: HTML/CSS/JS  
Backend: Flask API  
Database: PostgreSQL (Supabase)

## Setup
- Clone repo
- Add .env
- pip install -r requirements.txt
- python app.py

## API Endpoints
GET /health  
GET /profile  
GET /skills  
GET /skills/top  
GET /projects  
GET /projects/<skill>  

## Database Schema
profile, skills, projects
